== Description ==

BeamTheme Addons is a site specific plugin that adds features and functionality to sites using [Beam WordPress Theme](https://wordpress.org/themes/beam/ "Beam Theme").

After installation, [Beam Recent Posts Widget](https://www.beamtheme.com/beamtheme-addons/ "More Info") will show up
in the Widgets panel. Drag it to Sidebar to activate it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress
